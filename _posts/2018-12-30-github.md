###프로젝트 시작하기

```
$ git init
$ git add .
$ git commit -m "message"
$ git remote add origin 주소
$ git push -u origin master
```

###프로젝트 가져오기

```
$ cd 폴더 위치
$ git clone 주소
```

###Git 초기 설정
```
$ git config --global user.name "YOUR NAME"
$ git config --global user.email "you@email.com"
```

- 수정이 필요한 경우 
```
$ git commit --amend --reset-author
```

----------
#####기타 명령어
```
$ git status
$ git help
```
